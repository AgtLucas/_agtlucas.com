2º dia (03/06/2016)

Depois de um primeiro dia bem agitado por conta do check-in/credenciamento e entrega dos brindes, o segundo dia foi bem mais tranquilo…

PS.: A foto acima foi antes do evento começar. :)

Evolving the Visual Programming Environment with React

Nesta palestra, Jonas Gebhardt demonstra através de uma ferramenta de prototipagem bem legal como podemos tornar o ambiente de programação mais acessivo.

React Native Retrospective

Este ano (2016) React Native completou 1 ano de vida e Bonnie Eisenman apresenta uma retrospectiva do mesmo. Polêmico desde o seu anúncio e lançamento em 2015, onde na época em questão tinha suporte apenas para iOS, passando até os dias atuais onde conta com suporte para Android e o que o futuro nos reserva com o recente anúncio da parceria com a Microsoft e Samsung para fazer uso do RN em seus respectivos devices e plataformas.

The Evolution of React UI Development

Uma das palestras mais animadas do dia, Max Stoiber e Nik Graf fazem um review sobre o desenvolvimento de UI com React e apresentam seu mais novo projeto; Carte Blanche, que nada mais é que um espaço isolado de desenvolvimento que permite você desenvolver e testar componentes de uma maneira bem interessante. Vale a pena conferir!

Recomposing your React application

Andrew Clark discute diversas estratégias de como construir aplicações React preparadas para escalabilidade e reuso. Dica: Recompose.

Falcor: One Model Everywhere

Jafar Husain apresenta e detalha de uma maneira muito clara e objetiva Falcor, criado pelo Netflix. Nesta palestra ele compara o Falcor com o GraphQL explicando suas principais diferenças e quando usar qual.

JavaScript, React Native and Performance

Tadeu Zagallo e toda a equipe que trabalha no React Native têm buscado constantemente melhorar sua performance, tanto que existe uma equipe trabalhando apenas em otimizações. Nesta palestra ele explica o que têm sendo feito, desde forks no JSCore para adicionar otimizações específicas até otimizações nativas, como por exemplo, usufruir do multi-thread na inicialização das aplicações que fazem uso do React Native.

Building li.st for Android with Exponent and React Native

Membro do core team do React Native, Brent Vatne conta sua experiência de como foi desenvolver uma aplicação de grande porte para Android usando React Native; li.st. Muitas dicas valiosas!

GraphQL Future

O que esperar do GraphQL? Quais são os planos dos mantedores do projeto? Como o Facebook está usando o GraphQL? Estas e outras perguntas são respondidas por Laney Kuenzel e Lee Byron.

Building native mobile apps with GraphQL

Muitas, mas muitas vezes mesmo o GraphQL é confundido como uma biblioteca para ser usada apenas com React. O que não é verdade. Martijn Walraven demonstra como usar GraphQL num app Swift.

Q&A with core team members

Uma longa sessão de perguntas e respostas. :)

E assim acabou mais um ReactEurope, de longe o melhor evento de tecnologia que participei, muito networking, muitas amizades novas e muito conhecimento adquirido!
